About This Project

Research Context

This application supports Project B05 "Mark's Narrative Pneumatology" from the Collaborative Research Centre on "Narrating Ideas: Ideational Narratives in Cultural Transfers."

Methodology

The investigation combines narratological analysis with cognitive science to understand how Mark constructs the Holy Spirit as a background character through attentional cues rather than explicit representation.

Attentional Cues

  • Primacy Effect: Early mentions that establish character importance
  • Causal Implication: Events attributed to off-stage characters
  • Focalization Shifts: Changes in narrative perspective
  • Conspicuous Absence: Notable omissions or gaps
  • Prolepsis: Forward references anticipating character action

Technical Implementation

Built with vanilla JavaScript and D3.js for visualizations. Uses AI-powered analysis to identify narrative patterns in the Greek text.
